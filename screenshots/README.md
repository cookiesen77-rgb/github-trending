# Screenshots for GitHub Trending Project

This directory contains screenshots of the application interface.

## Screenshot Guidelines

When adding screenshots, please follow these guidelines:

1. **File Naming:**
   - `cli-screenshot.png` - CLI interface screenshot
   - `web-daily.png` - Web interface showing daily trending
   - `web-weekly.png` - Web interface showing weekly trending
   - `web-monthly.png` - Web interface showing monthly trending

2. **Image Quality:**
   - Use PNG format for better quality
   - Keep file sizes reasonable (< 2MB per image)
   - Ensure screenshots are clear and readable

3. **Content:**
   - Show real data from the application
   - Include both light and dark theme variants if applicable
   - Demonstrate key features and functionality
